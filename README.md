# Favour Capital Website

Rebuild of favour-capital.com. Next.js (App Router) + TypeScript + Tailwind CSS + Framer Motion, deployed on Vercel.

See [`DESIGN_SYSTEM.md`](./DESIGN_SYSTEM.md) for tokens/components and [`SCOPE.md`](./SCOPE.md) for pages/content/phases. Session rules for agents working on this repo are in [`CLAUDE.md`](./CLAUDE.md).

## Development

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Checks

```bash
pnpm lint
pnpm typecheck
pnpm format:check
pnpm build
```
